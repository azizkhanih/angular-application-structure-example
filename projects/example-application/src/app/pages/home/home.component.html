Dynamic Form Demo

  • To review the dynamic form result you have to select Dynamic Form link from the top navigation menu.

    (This demo will support Email format validation too.)

  • You redirect to DynamicFormExampleComponent that is responsible for getting data from JSON file and passing DynamicFields list as a parameter to DynamicFormBuilderComponent .

  • DynamicFormExampleService is created for mapping JSON object to different types like InputText or InputCheckbox that are extended form DynamicField base class.

  • All created components and classes and models for creating dynamic form are located at shared\components\dynamic-form:

    • dynamic-form.module
    • dynamic-form-builder
    • dynamic-field-builder
    • field-type
    • dynamic-field.model
    • dynamic-field.type.enum
    • dynamic-form-control.service

  • DynamicFieldBuilderControlService is responsible for creating FormGroup from DynamicField object.

  • DynamicFormBuilderComponent is responsible for creating dynamic from.

  • DynamicFieldBuilderComponent is responsible for creating form-group and set dynamic validations and errors.
